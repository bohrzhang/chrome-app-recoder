# wukong-chrome-plugin
一款用来分维度记录URL链接的Chrome插件

## 使用
1. 应用商城搜索 **悟空记录** ，点击 **添加至** Chrome
2. 访问网址，点击插件icon，填写相应信息并保存
3. 打开列表，选中组合字段后点击打开
4. 点击 **扩展程序选项** 可打开后台管理页面
